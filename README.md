
# Titanic Survival Prediction

## Problem
Predict which passengers survived the Titanic disaster using machine learning.

## Approach
- Exploratory Data Analysis on 891 records
- Feature engineering: Age imputation, encoding Sex/Embarked
- Model: Logistic Regression / Random Forest
- Evaluated using accuracy and confusion matrix

## Results
Achieved ~80% accuracy on test data

## Tools
Python | Pandas | Scikit-learn | Matplotlib | Jupyter Notebook
